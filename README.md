# Project Overview

LearnMate is an AI-powered career and learning assistant developed using IBM watsonx.ai and IBM Granite foundation models.

The assistant helps students:

- Generate personalized learning roadmaps
- Explore AI/ML career paths
- Receive skill recommendations
- Discover relevant learning resources
- Build structured career development plans

The system leverages Agentic AI concepts to provide intelligent and adaptive responses.
